# Get-SeKeys

## SYNOPSIS
Return a list of the available special keys

## SYNTAX

```
Get-SeKeys [<CommonParameters>]
```

## DESCRIPTION
Return a list of the available special keys.
These keys can be used with `Send-SeKeys` by using their name in the following format `{{SpecialKeyName}}`

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-SeKeys
```
